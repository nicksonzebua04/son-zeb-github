Halo, Perkenalkan nama saya Nickson N.F Zebua, mahasiswa semester 2 Universitas Atma Jaya Yogyakarta saya dari Fakultas Teknologi Industri prodi Informatika.

"Saat ini saya sedang fokus mengembangkan pemahaman mendalam tentang keamanan jaringan, etika peretasan (ethical hacking), dan perlindungan data.
